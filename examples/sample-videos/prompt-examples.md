## 提示词结构模板

```
[时长]秒竖屏短视频，小区茶店「小拾山茶事」，[风格]，
[开头3秒钩子]，
[中景主体内容]，
[特写细节]，
[背景环境]，
结尾字幕"[行动号召文案]"，
添加[背景音乐类型]，字幕清晰，色彩温暖，4K高清，电影颗粒感，
画面稳定，光影柔和，细节丰富，生活化纪实风格
```

## 参数说明

| 参数 | 可选值 |
|------|--------|
| 时长 | 15-30秒 |
| 风格 | 暖光温馨/清新自然/国风雅致 |
| 开头钩子 | 特写推门/特写茶杯/特写人物表情 |
| 中景 | 泡茶/品茶/文创展示/邻里聊天 |
| 特写 | 茶杯热气/茶叶形态/文创细节 |
| 背景 | 手绘茶画/扎染挂件/茶罐博古架 |
| 背景音乐 | 舒缓吉他/轻柔古筝/温馨轻音乐 |
